Welcome.

For Scatter Plots:

  • New Approach: Visualization of parallel 2d hidden layer neurons.
  • PCA: Principal Component Analysis of the last layer of a model (No 2d parallel output only main part).
  • TSNE: t-SNE visualizations of the last layer of the model (No 2d parallel output only main part).
  • Accuracies: View the accuracy metrics for the dataset.

For Accuracy Plots:

  • model accuracy: Accuracy of the model without any additional low dimensional part for reference.
  • new model accuracy: Accuracy of the main part of the parallel output model.
  • vis model accuracy: Accuracy of the low dimensional (2d) part of the parallel output model.
